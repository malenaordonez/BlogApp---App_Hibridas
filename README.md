El nombre del proyecto es BlogApp
BlogApp es una app donde el usuario puede crear sus propios posteos y leer los posteos de los demas. Cuenta con un CRUD completo de posteos. Hay vistas públicas (los posteos de todos los usuarios) y privadas (panel de admin).
